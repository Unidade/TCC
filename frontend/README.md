# Frontend - TCC Interview Simulator

React + TypeScript frontend with 3D avatar and chat interface.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Features

- **3D Avatar**: Ready Player Me avatar with lip-sync animation
- **Chat Interface**: Clean, modern chat UI using shadcn/ui
- **WebSocket Communication**: Real-time communication with backend
- **Audio Playback**: Synchronized audio with lip-sync animation

## Tech Stack

- React 19
- TypeScript
- Vite
- React Three Fiber (3D rendering)
- shadcn/ui (UI components)
- Tailwind CSS
